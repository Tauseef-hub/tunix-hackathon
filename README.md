# Tunix Hackathon: Teaching Gemma to Reason

**Competition:** Google Tunix Hack - Train a Model to Show Its Work  
**Prize Pool:** $100,000  
**Deadline:** January 12, 2026

## Approach

1. Use free datasets (GSM8K, MATH) with existing reasoning traces
2. Reformat into competition format (`<thinking>`, `<answer>` tags)
3. Fine-tune Gemma using Tunix GRPO on Kaggle TPU
